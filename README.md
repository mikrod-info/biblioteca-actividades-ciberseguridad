Biblioteca abierta de actividades educativas orientadas a promover el uso seguro, responsable y crítico de las tecnologías digitales.

El proyecto reúne materiales desarrollados e implementados en actividades de extensión universitaria vinculadas con alfabetización digital, ciudadanía digital y ciberseguridad, principalmente con grupos de personas adultas mayores.

## Objetivo

Compartir actividades, recursos y experiencias que puedan ser reutilizados, adaptados o ampliados en talleres, cursos y espacios educativos relacionados con:

* Internet y seguridad digital.
* Contraseñas y autenticación.
* Inteligencia Artificial.
* Ciudadanía digital.

Las propuestas priorizan la participación, el intercambio de experiencias y la comprensión de conceptos mediante ejemplos y actividades prácticas.

## Tecnologías utilizadas

* Eleventy (11ty)
* Pico CSS
* Docker
* GitHub Actions
* Markdown

## Desarrollo

Este sitio genera contenido estático a partir de archivos Markdown organizados por actividad, facilitando la documentación, mantenimiento y publicación de materiales educativos.
